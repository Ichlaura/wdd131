Paola's English School

Website Development Plan

Site Name:

Paola's English School

Actual Domain: paolasenglishschool.website

Location: Togane, Chiba, Japan

The brand name establishes personal connection with international students while clearly communicating our core service. The domain provides consistent branding for our online English education platform.

Site Purpose & Content

Home Page

School introduction, mission statement, and key differentiators (native teachers, flexible scheduling)

Courses

Program details: Conversational English, Business English, Exam Preparation (TOEFL/IELTS)

Teachers

Native speaker profiles with credentials, teaching experience, and specialties

Testimonials

Success stories from students in Colombia, Chile, Japan, and other regions

Registration

Pricing options, schedule converter (JST to local time), enrollment form

Contact

Inquiry form, physical location (Togane, Japan), social media links

Multilingual Support: Spanish, Japanese, Portuguese, Chinese

User Scenarios

  1. "As a Japanese student, how can I verify the teachers' qualifications?"
    → Addressed in Teacher Profiles section with credential verification
  2. "What time would classes be in Colombia if they're at 7am Japan time?"
    → Interactive timezone converter in Registration section
  3. "Can I see examples of student progress before enrolling?"
    → Testimonials section with before/after language assessments
  4. "Is there mobile app support for classes?"
    → Technology requirements explained in Courses section

Typography

Inter Typeface / 日本語フォント例（Noto Sans JP）:

  • Bold 700: Headings and primary CTAs
  • Semi-bold 600: Subheadings and buttons
  • Medium 500: Highlighted text
  • Regular 400: Body text and paragraphs
  • ボールド 700：見出しと主なコールトゥアクション
  • セミボールド 600：小見出しとボタン
  • ミディアム 500：強調されたテキスト
  • レギュラー 400：本文や段落

Technical Implementation

HTML/CSS Requirements

  • Semantic HTML5 structure
  • Responsive grid and flexbox layouts
  • CSS variables for consistent theming
  • Optimized images (WebP format where possible)
  • Lazy loading for images below the fold

JavaScript Features

  • Language switcher with localStorage persistence
  • Interactive timezone converter
  • Form validation for registration
  • Testimonial carousel/slider
  • Mobile menu functionality

Pages Structure

  1. index.html - Homepage
  2. courses.html - Programs offered
  3. teachers.html - Instructor profiles
  4. testimonials.html - Student success stories
  5. register.html - Enrollment form
  6. contact.html - Contact information
  7. references.html - Citation page (unstyled)
